网页项目展示

在这里展示你的三个网页项目

添加网页项目

拖放网页文件夹到这里或点击上传 (此功能已禁用，请通过 GitHub 仓库手动管理)

手动添加步骤：

  1. 将你的网页项目文件夹 (包含 `index.html` 及其他文件) 复制到 GitHub 仓库的 `/webpages/` 目录下。
  2. 编辑你的 `resources.json` 文件，添加一个新条目，`content` 路径应指向该网页项目的主 HTML 文件，例如：`/my-resource-center/webpages/你的项目名/index.html`。
  3. 将更改提交并推送到 GitHub 仓库的 `main` 分支。

上传文档

拖放文件到这里或点击上传 (此功能已禁用，请通过 GitHub 仓库手动管理)

手动添加步骤：

  1. 将文档文件复制到 GitHub 仓库的 `/documents/` 目录下。
  2. 编辑你的 `resources.json` 文件，添加一个新条目，`content` 路径应指向该文档文件，例如：`/my-resource-center/documents/你的文档名.pdf`。
  3. 将更改提交并推送到 GitHub 仓库的 `main` 分支。

图片与视频

上传媒体文件

拖放图片或视频到这里或点击上传 (此功能已禁用，请通过 GitHub 仓库手动管理)

手动添加步骤：

  1. 将图片或视频文件复制到 GitHub 仓库的 `/media/` 目录下。
  2. 编辑你的 `resources.json` 文件，添加一个新条目，`content` 路径应指向该媒体文件，例如：`/my-resource-center/media/你的图片或视频名.jpg`。
  3. 将更改提交并推送到 GitHub 仓库的 `main` 分支。

重要链接

保存和访问课程相关的重要链接

添加新链接 (此功能已禁用，请通过 GitHub 仓库手动管理)

手动添加步骤：

  1. 编辑你的 `resources.json` 文件，在 `links` 数组中添加新的链接对象。
  2. 提供 `id`、`name` (标题)、`url` 和 `date` 字段。
  3. 将更改提交并推送到 GitHub 仓库的 `main` 分支。
